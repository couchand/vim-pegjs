vim-pegjs
=========

vim syntax highlighting for pegjs grammar files

 * introduction
 * installation
 * more information

introduction
------------

pegjs grammar files have a simple, expressive syntax.  highlight it.

installation
------------

    git clone https://github.com/couchand/vim-pegjs.git
    cp vim-pegjs/* ~/.vim/ -r

more information
----------------
